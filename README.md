# 🤖 Rule-Based Chatbot

## 📌 Description

This project is a simple rule-based chatbot developed using Python. It responds to predefined user inputs such as greetings, common questions, time requests, and exit commands.

## ✨ Features

- Responds to greetings
- Answers predefined questions
- Displays current system time
- User-friendly console interaction
- Exit option using "bye"

## 🛠 Technologies Used

- Python 3

## ▶️ How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run:

```bash
python chatbot.py
```

## 📷 Sample Output

```
Welcome to AI ChatBot!

You: hello
Bot: Hello! How can I help you?

You: time
Bot: 15:30:45

You: bye
Bot: Goodbye! Have a nice day.
```
